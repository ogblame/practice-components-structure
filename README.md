# React Component Practice

Учебный проект, посвященный практике компонентного подхода в React.

## Что было реализовано

- Разбиение интерфейса на независимые компоненты
- Передача данных через props
- Отрисовка списков с использованием `map`
- Использование моковых данных
- Практика адаптивной верстки
- Организация структуры проекта и компонентов

## Технологии

- React
- TypeScript
- CSS
- Vite

## Цель проекта

Закрепить основы разработки на React, научиться правильно разбивать интерфейс на компоненты и организовывать структуру проекта.
